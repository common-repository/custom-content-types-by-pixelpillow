== Description ==

Create posts based on content type in a Tumblr like way. We've included a set of default icons for most thinkable content types, or just add your own custom content types with your own icons. The icons are also shown in the post overview for optimal visual distinction between posts.

*We recommend using the Advance Custom Fields plugin to create custom field per content type* 

== Installation ==

1. Upload the 'pixelpillow-custom-content-types' directory to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
